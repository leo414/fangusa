# fangusa

> A web app for fangusa.

## Build Setup

``` bash
# install dependencies
npm install

# serve with hot reload at localhost:3000
npm start

# build for production with minification
npm run build

# build for production and view the bundle analyzer report
npm run build --report
```

### Technology stack
ES6/ES7
vue 2.x
vue-resource
vue-router
vuex
iview
Sass
